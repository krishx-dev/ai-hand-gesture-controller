# 🚀 AI Hand Gesture Controller (Gesture Engine OS)

> A real-time AI-powered gesture recognition system that enables touchless interaction with your computer using hand movements.

---

## 🧠 Overview

**Gesture Engine OS** is a modern, real-time hand gesture control system built with computer vision and a sleek glassmorphism UI. It uses your webcam to detect hand gestures and converts them into actionable system commands.

This project demonstrates the fusion of **AI + UI/UX + real-time processing** to create a futuristic interaction experience.

---

## ✨ Features

- 🎥 Live Camera Feed  
- ✋ Real-time Hand Tracking  
- 🧠 Gesture Recognition Engine  
- ⚡ Instant System Actions  
- 📊 Confidence Metrics & Analytics  
- 🗂 Gesture History Logging  
- 🧭 Gesture Mapping Dashboard  
- 🎨 Modern Glassmorphism UI  
- 🌙 Dark Themed Interface  
- ⚙️ Modular Component Architecture  

---

## 🎮 Supported Gestures

| Gesture | Action |
|--------|--------|
| 👍 Thumbs Up | Volume Up |
| 👎 Thumbs Down | Volume Down |
| ✋ Open Palm | Play / Pause |
| ✊ Closed Fist | Take Screenshot |
| ☝️ Pointing | Click / Select |
| ✌️ Two Fingers | Scroll |
| 👌 OK Sign | Open Browser |
| 👉 Swipe Left | Previous Track |
